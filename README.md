## üöÄ Getting Started

### Prerequisites

You will need an API key from [OpenWeatherMap](https://openweathermap.org/api).

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/nived-vijay/Weather-Where.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Weather-Where
   ```
3. Open `script.js` and replace the placeholder API key with your own:
   ```javascript
   const API_KEY = 'YOUR_API_KEY_HERE';
   ```
4. Open `index.html` in your browser to launch the app!
